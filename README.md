## Usage

Define the following colors:
```
$hbm-color-text: #XXXXXX;
$hbm-color-highlight: #XXXXXX;
$hbm-color-secondary: #XXXXXX;
```

Optionally define the following maps:
```
$toggable-items: ();
$table-cell-widths: (10, 25, 50, 75, 100, 125, 150, 200, 250);
```
